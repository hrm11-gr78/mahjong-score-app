# GitHubへの変更の反映方法 (更新手順)

手元のパソコンで行った変更（削除機能の追加など）を、GitHub上のリポジトリに反映させる手順です。

## 方法A: コマンドラインを使う場合（推奨）
Gitのセットアップが済んでいる場合は、以下のコマンドをターミナルで実行するだけで更新できます。

1. ターミナル（PowerShellなど）を開き、プロジェクトのフォルダに移動します。
   ```powershell
   cd c:\Users\hirom\OneDrive\Desktop\麻雀スコア管理
   ```

2. 以下の3つのコマンドを順番に実行します。
   ```bash
   git add .
   git commit -m "削除機能を追加"
   git push
   ```

   - `git add .`: 変更された全てのファイルを準備エリアに追加します。
   - `git commit -m "..."`: 変更内容にメッセージを付けて保存します。
   - `git push`: GitHubに変更を送信します。

## 方法B: ブラウザを使う場合
コマンドラインが難しい場合は、ブラウザから直接ファイルをアップロードして上書きすることも可能です。

1. GitHubのリポジトリページを開きます。
2. 「Add file」 > 「Upload files」を選択します。
3. パソコンのフォルダにある、**変更したファイル**（今回は `js/app.js`, `js/storage.js`, `index.html` など）をドラッグ＆ドロップします。
   ※ 面倒な場合は、フォルダの中身を全部ドラッグ＆ドロップしても大丈夫です（変更がないファイルは無視されます）。
4. 下の「Commit changes」ボタンをクリックします。

## 反映の確認
更新後、数分待ってからスマホでアプリを開き直すと、新しい機能（削除ボタンなど）が使えるようになります。
反映されない場合は、アプリを再読み込みしてみてください。
